Addendum

  • Implemented with the use of Holy Grail Layout, the given calculator tool allows the user to perform addition, subtraction, multiplication and division, as well as other elementary operations.
  • To handle user input it has dynamic javascript functions which include addition of number and an operator, calculation of the result and clear the display.
  • The right sidebar contains links to external products, and is responsive across devices due to CSS Grid. The footer of the page contains the last changed date and it has been printed by using JavaScript.
  • Applications such as input validation, handling of JavaScript events, and dynamic CSS style of management were used to deal various problems including validation of the input accepted, restriction for numbers of operator sign consecutively, and fluidity in as many term sizes as possible.
  • This solution presents a clean and useful tool that is user friendly and driven more so by simple, tab based interfaces.
